## ⚙️ 1. 기본 압축 및 모델 분류 용어

### ⚖️ 1) 양자화 (Quantization)
* **쉬운 비유**: **"해상도 줄이기"** 또는 **"수능 책 요약본 만들기"**
* **개념 설명**: AI 모델은 컴퓨터가 소수점(FP16)으로 계산을 해서 덩치가 매우 큽니다. 이 숫자를 단순한 정수(INT4, INT2 등)로 거칠게 깎아서 모델의 크기와 메모리 용량을 극단적으로 줄이는 작업입니다.
* **비즈니스 가치**: 
  * 4비트(W4)로 깎으면 크기가 1/4로 줄고 성능 손실이 거의 없어 필수 관문입니다.
  * 2비트(W2)로 깎으면 크기는 엄청 작아져 스마트폰에 쏙 들어가지만, 정보가 너무 많이 깨져서 모델이 바보가 됩니다. (이를 되살리기 위해 Recover-LoRA 같은 기법이 필요합니다.)

### 🧠 2) SLM (Small Language Model, 소형 언어 모델)
* **쉬운 비유**: **"특정 분야만 파고든 전문 족집게 과외 선생님"**
* **개념 설명**: ChatGPT나 Gemini 같은 모델은 모든 지식을 다 아는 거대 언어 모델(LLM)인 반면, 1B(10억 개)~8B(80억 개) 파라미터 수준으로 덩치를 가볍게 줄여 특정 도메인(예: 세무, 니치 판정 등)에만 특화시킨 가벼운 소형 모델입니다.
* **비즈니스 가치**: 클라우드 API 호출비 없이 개인 PC나 스마트폰(온디바이스) 로컬에서 무료로 초고속 구동이 가능합니다.

---

## 🛠️ 2. 파인튜닝(미세조정) 및 학습 용어

### 🎀 3) LoRA (Low-Rank Adaptation)
* **쉬운 비유**: **"이미 지어진 건물 위에 임시 옥탑방 하나 얹어서 쓰기"**
* **개념 설명**: 수십억 개의 모델 전체 뇌세포(가중치)를 학습시키는 것은 GPU 비용이 천문학적으로 듭니다. 대신, 원본 모델은 꽁꽁 얼려두고(Freeze), 그 옆에 아주 가볍고 얇은 **LoRA 어댑터(신경망 레이어)**를 덧붙여서 이 얇은 어댑터만 빠르게 학습시키는 기법입니다.
* **비즈니스 가치**: 전체 재학습 대비 GPU 비용과 시간을 95% 이상 절감하여, 1인 창업가가 단돈 몇 천 원에 파인튜닝을 할 수 있게 만든 일등 공신입니다.

### 👑 4) DoRA (Weight-Decomposed Low-Rank Adaptation)
* **쉬운 비유**: **"목소리 볼륨(크기) 조절과 말투(방향) 교정을 분리해서 사투리만 싹 고치기"**
* **개념 설명**: LoRA의 직계 후속작이자 상위 호환 기술입니다. 학습할 가중치를 **크기(Magnitude)**와 **방향(Direction)** 성분으로 칼같이 분해한 뒤, 크기 성분은 가볍게 맞추고 방향 성분에만 LoRA를 적용해 정밀 튜닝합니다.
* **비즈니스 가치**: LoRA만큼 가볍고 용량을 안 먹으면서도, 모델 전체를 통째로 다 학습시킨 것(Full Finetuning)에 육박하는 압도적인 정확도를 보장합니다. 특히 깨진 2비트 모델 성능 복구에 최강입니다.

### ⚡ 5) QLoRA / QDoRA (Quantized LoRA / Quantized DoRA)
* **쉬운 비유**: **"바보가 된 양자화 모델에게 DoRA/LoRA 어댑터라는 보청기 끼워주기"**
* **개념 설명**: 베이스 모델을 이미 4비트나 2비트로 깎아놓은(Quantized) 초경량 상태에서, 그 위에 LoRA나 DoRA 어댑터만 얹어서 훈련시키는 기법입니다.
* **비즈니스 가치**: GPU 메모리를 극단적으로 적게 쓰기 때문에, 일반 가정용 그래픽 카드 1장으로도 훌륭한 파인튜닝을 가능하게 해 줍니다.

## 🧪 3. 데이터 및 학습 메커니즘 용어

### 🧪 6) 지식 증류 (Knowledge Distillation)
* **쉬운 비유**: **"대기업 수석 연구원(Teacher)의 노하우를 인턴 사원(Student)에게 1:1로 족집게 전수하기"**
* **개념 설명**: 아주 큰 고성능 모델(Teacher, 예: 제미나이)이 내뱉는 정답의 확률 분포(Logit)를 보고, 작은 소형 모델(Student)이 그 생각의 흐름과 확률을 그대로 모방하여 닮아가도록 훈련하는 기술입니다.
* **비즈니스 가치**: 작은 모델의 뇌 용량 한계 속에서도 대형 모델의 똑똑함을 가장 빠르게 복제할 수 있는 핵심 기법입니다.

### 🧪 7) 합성 데이터 (Synthetic Data)
* **쉬운 비유**: **"인공지능이 스스로 소설을 써서 만든 가짜 학습지"**
* **개념 설명**: 사람이 손수 작성한 데이터가 아니라, LLM(대형 언어 모델)에게 프롬프트를 주어 스스로 무수히 많은 가상의 비즈니스 시나리오나 문제 세트를 만들어내게 한 데이터입니다.
* **비즈니스 가치**: 데이터 수집 비용이 **0원**에 수렴하며, 저작권이나 개인정보 보호법 위반 리스크로부터 완전히 자유로운 깨끗한 학습용 데이터를 1초 만에 무한 생성할 수 있습니다.

### 📚 8) SFT (Supervised Fine-Tuning, 지도 미세조정)
* **쉬운 비유**: **"문답형 족보(Q&A)를 달달 외우게 학습시키기"**
* **개념 설명**: AI에게 [질문 - 답변] 쌍의 정답 데이터를 주고, 그 스타일과 지식을 그대로 복사해서 대답하도록 훈련시키는 가장 대중적이고 기본적인 미세조정 학습 방식입니다.
